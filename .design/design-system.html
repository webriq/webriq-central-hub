Voice & Tone
Rules for all UI copy written in this app.
Casing
Title Case — page headings, card titles, section names
Sentence case — body copy, descriptions, placeholder text
ALL CAPS — eyebrow labels, sidebar group headers (MAIN, DEVELOPER, ADMIN)
monospace — all IDs and codes: WRQ-001, WRQ-CLIENT-1234, ZP-10293
Numbers & Time
Always use numerals: "5 overdue", "24 active", "87%", "3h 45m"
Relative timestamps: "2m ago", "1h ago", "1d ago"
Due dates: "May 15", "Jun 1" (no year unless different year)
Button / CTA Copy
Use inline arrow: "Continue →", "View all →", "New Project →"
Use inline checkmark for submit: "Submit Onboarding ✓"
Never use exclamation points in primary copy
Back/cancel buttons: "← Back", "Cancel" (no arrow)
